# Manifest/清单
* https://www.scala-lang.org/api/2.13.16/scala/reflect/Manifest.html

A `Manifest[T]` is an opaque descriptor for type `T`. Its supported use is to give access to the erasure of the type as a `Class` instance, as is necessary for the creation of native `Array`s if the class is not known at compile time.
