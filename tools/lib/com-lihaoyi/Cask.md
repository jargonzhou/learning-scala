# Cask
* https://github.com/com-lihaoyi/cask
* [The Scala Toolkit - Building web servers with Cask](https://docs.scala-lang.org/toolkit/web-server-intro.html)

> Cask: a Scala HTTP micro-framework. 
>
> Cask is a simple Scala web framework inspired by Python's Flask project. 
> It aims to bring simplicity, flexibility and ease-of-use to Scala webservers, avoiding cryptic DSLs or complicated asynchrony. 
> Cask makes it easy to set up a website, backend server, or REST API using Scala
